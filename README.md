# CodeAlpha Stock Portfolio Tracker

## Description
A simple Stock Portfolio Tracker developed in Python as part of the CodeAlpha Python Programming Internship. This project helps users track their stock investments and calculate the total value of their portfolio.

## Features
- Add stock names and quantities
- Calculate total portfolio value
- Display portfolio details
- Simple command-line interface
- Beginner-friendly Python project

## Technologies Used
- Python

## How to Run
1. Download the project.
2. Open the project folder.
3. Run the Python file:
   python stock_portfolio_tracker.py
4. Enter stock details and view the portfolio value.
